一個關於台灣蝴蝶食草與蜜源植物的圖鑑，目前尚有對蜜源植物的相關資訊  
**何謂蝴蝶食草？**  
　　每種蝴蝶的幼蟲對食物的選擇都有專一性，即每種蝴蝶各以特定的若干種植物為食物，此稱為食草。  
**何謂蜜源植物？**  
　　會產生花蜜的植物，能夠吸引蜜蜂、蝴蝶等昆蟲，蜜源植物幾乎一年四季開花，吸引昆蟲前來覓食。  
　　有些食草植物在開花期也能成為蝶類的蜜源植物，如柑橘類、馬利筋等。
